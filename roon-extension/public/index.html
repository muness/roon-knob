What is this?

Roon Knob is a DIY hardware volume controller for Roon. Features a rotary encoder, touchscreen display, and WiFi connectivity to control your Roon zones.

Getting Started

1. Flash firmware using the web flasher (Chrome/Edge)
2. Connect to WiFi via captive portal
3. Run the bridge via Docker
4. Authorize in Roon Settings → Extensions
